## Overview

This workflow automates the detection of abnormal spreads across multiple liquidity providers (LPs) using a machine learning model. By continuously analyzing spread data, you can identify pricing errors, latency issues, or manipulation attempts in real-time.

## Prerequisites

- Python 3.8+ installed
- Access to LP APIs (e.g., FXCM, LMAX, Integral) or a data feed (e.g., Bloomberg, Reuters)
- Basic knowledge of machine learning and time-series analysis
- Cloud or local server for running the model (e.g., AWS EC2, GCP)

## Step 1: Collect Spread Data from LPs

Write a Python script to pull bid/ask prices from each LP at regular intervals (e.g., every 100ms). Store the spread (ask - bid) in a time-series database.

```python
import requests
import time
from influxdb import InfluxDBClient

client = InfluxDBClient(host='localhost', port=8086)
client.switch_database('spreads')

def fetch_spread(lp_url):
    response = requests.get(lp_url)
    data = response.json()
    spread = data['ask'] - data['bid']
    return spread

while True:
    for lp in lp_list:
        spread = fetch_spread(lp['url'])
        json_body = [
            {
                "measurement": "spread",
                "tags": {
                    "lp": lp['name']
                },
                "fields": {
                    "value": spread
                }
            }
        ]
        client.write_points(json_body)
    time.sleep(0.1)
```

## Step 2: Build an Anomaly Detection Model

Use TensorFlow to create an autoencoder that learns normal spread patterns. Train it on historical data (e.g., 30 days).

```python
import tensorflow as tf
from tensorflow import keras
import numpy as np

# Load historical spread data (shape: samples, timesteps, features)
X_train = np.load('spread_history.npy')

model = keras.Sequential([
    keras.layers.LSTM(64, activation='relu', input_shape=(X_train.shape[1], X_train.shape[2])),
    keras.layers.RepeatVector(X_train.shape[1]),
    keras.layers.LSTM(64, activation='relu', return_sequences=True),
    keras.layers.TimeDistributed(keras.layers.Dense(X_train.shape[2]))
])
model.compile(optimizer='adam', loss='mse')
model.fit(X_train, X_train, epochs=50, batch_size=32, validation_split=0.2)
model.save('anomaly_detector.h5')
```

def get_recent_data(lp_name, window=100):
    query = f"SELECT last(value) FROM spread WHERE lp='{lp_name}' ORDER BY time DESC LIMIT {window}"
    result = client.query(query)
    points = list(result.get_points())
    values = [p['value'] for p in reversed(points)]
    return np.array(values).reshape(1, -1, 1)
