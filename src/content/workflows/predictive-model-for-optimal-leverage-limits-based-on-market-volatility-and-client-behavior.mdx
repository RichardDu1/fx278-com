## Overview

This workflow guides risk managers through building a predictive model that sets optimal leverage limits per client based on real-time market volatility and individual trading behavior. The model uses historical volatility indices (e.g., VIX) and client metrics (e.g., win rate, max drawdown) to output a leverage multiplier.

## Prerequisites

- Python 3.8+ installed
- Access to market volatility data (e.g., VIX API or CSV)
- Client trading history (CSV with columns: client_id, trade_date, pnl, leverage_used, win_loss_flag)
- Basic familiarity with Python, pandas, and scikit-learn

## Step 1: Data Collection and Preparation

1. **Collect Market Volatility Data**
   - Download daily VIX closing prices from Yahoo Finance or FRED.
   - Save as `vix.csv` with columns: `date`, `vix_close`.

2. **Collect Client Behavior Data**
   - Export client trading history from your CRM or trading platform.
   - Ensure CSV includes: `client_id`, `trade_date`, `pnl`, `leverage_used`, `win_loss_flag` (1 for win, 0 for loss).

3. **Merge Datasets**
   - Use pandas to merge client trades with VIX data on `trade_date`.
   - Add derived features: `rolling_volatility_30d` (30-day rolling average of VIX), `client_win_rate_30d` (30-day rolling win rate per client), `client_max_drawdown_30d` (30-day rolling max drawdown).

```python
import pandas as pd

vix = pd.read_csv('vix.csv', parse_dates=['date'])
clients = pd.read_csv('client_trades.csv', parse_dates=['trade_date'])
merged = pd.merge(clients, vix, left_on='trade_date', right_on='date', how='left')

# Sort by client and date
merged.sort_values(['client_id', 'trade_date'], inplace=True)

# Rolling features
merged['rolling_volatility_30d'] = merged.groupby('client_id')['vix_close'].transform(lambda x: x.rolling(30, min_periods=1).mean())
merged['client_win_rate_30d'] = merged.groupby('client_id')['win_loss_flag'].transform(lambda x: x.rolling(30, min_periods=1).mean())
merged['client_max_drawdown_30d'] = merged.groupby('client_id')['pnl'].transform(lambda x: x.rolling(30, min_periods=1).apply(lambda y: (y.cummax() - y.cummin()).max()))
```

## Step 2: Define Target Variable

Define the optimal leverage limit as the maximum leverage that keeps the client's risk below a threshold (e.g., 2% daily VaR). For simplicity, use a rule: if the client's daily loss exceeds 2% of account equity, the leverage is too high. Create a binary target: `over_leveraged` (1 if loss > 2% of equity, else 0).

```python
# Assume account_equity column exists or calculate from initial balance
merged['daily_loss_pct'] = -merged['pnl'] / merged['account_equity']
merged['over_leveraged'] = (merged['daily_loss_pct'] > 0.02).astype(int)
```

## Step 3: Build Predictive Model

1. **Select Features**
   - `rolling_volatility_30d`, `client_win_rate_30d`, `client_max_drawdown_30d`, `leverage_used`.

2. **Train/Test Split**
   - Use time-based split: train on first 80% of dates, test on last 20%.

3. **Train a Classifier**
   - Use Random Forest from scikit-learn to predict `over_leveraged`.

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split

features = ['rolling_volatility_30d', 'client_win_rate_30d', 'client_max_drawdown_30d', 'leverage_used']
X = merged[features]
y = merged['over_leveraged']

# Time-based split
split_date = merged['trade_date'].quantile(0.8)
train = merged[merged['trade_date'] <= split_date]
test = merged[merged['trade_date'] > split_date]

X_train = train[features]
y_train = train['over_leveraged']
X_test = test[features]
y_test = test['over_leveraged']

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
```

## Step 4: Determine Optimal Leverage Limits

For each client and current market conditions, find the maximum leverage that yields a predicted probability of over-leverage below a threshold (e.g., 10%). Use a grid search over leverage values.

```python
import numpy as np

def optimal_leverage(client_row, model, threshold=0.1):
    leverages = np.arange(1, 101, 1)  # 1:1 to 100:1
    probs = []
    for lev in leverages:
        row = client_row.copy()
        row['leverage_used'] = lev
        prob = model.predict_proba(pd.DataFrame([row]))[0][1]
        probs.append(prob)
    # Find max leverage where prob < threshold
    for lev, prob in zip(leverages, probs):
        if prob < threshold:
            best = lev
        else:
            break
    return best

# Apply to latest data for each client
latest = merged.groupby('client_id').last().reset_index()
latest['optimal_leverage'] = latest.apply(lambda row: optimal_leverage(row, model), axis=1)
```
